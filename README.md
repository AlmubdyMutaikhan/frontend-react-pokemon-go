# Exciting, robust web app built in React.js about Pokemons

# Try it!
> https://my-pokemon-qwant-app.herokuapp.com/

## Description
> Robust web app providing big data about all pokemons in a glassphormism interface.

## How to navigate ?
* This web app has 3 sections:
    1. Main (Loads all pokemons using lazy load feature).
    2. About (Small description about this project).
    3. Search (Dynamic search bar of pokemons).

## Feautures
* Lazy load of components and images
* Responsive web app
* Dynamic search bar
* Fancy animation effects

 
## Local Setup

```sh
$ git clone https://github.com/AlmubdyMutaikhan/frontend-react-pokemon-go.git
```

```sh
$ cd frontend-react-pokemon-go
```

```sh
$ npm install
```

## Run Locally

```sh
$ npm start
```
